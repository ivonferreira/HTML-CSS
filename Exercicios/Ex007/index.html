Principais formatações

negrito / Destaque

Não Semantico

Nesta frase, temos um termo em negrito usando a tag <b> </b>

Semantico

Nesta frase, tenho um termo em destaque usando a tag <strong> </strong>

Itálico / Ênfase

Não Semantico

Nesta frase, temos um termo em italico usando a tag <i></i>

Semantico

Nesta frase, temos um termo em ênfase usando a tag <em></em>

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark></mark>

E no outro parágrafo , temos outro outro teste marcado no final

Texto Grande e Pequeno

Estamos criando um texto grande(tag <big></big>) e um texto pequeno(tag <small></small>)

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado com a tag <del></del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois com a tag <ins></ins>

Existe também o sublinhado com a tag <u></u>que não é semantico

Texto sobrescrito

Para escrever principalmente equações como x20+3 com a tag <sup></sup>

Texto subescrito

Para escrever principalmente formulas como H2O com a tag <sub></sub>
