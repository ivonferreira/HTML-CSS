Usando links externos

Para links ancora usamos a tag <a></a>, e os parametros target com valor _blank para abrir em uma nova aba. Ainda colocamos o parametro rel com valor external para sinalizar para o google que vai para um link externo.

Usando Links Internos

Faz ligação com uma pagina interna do nosso site. e para cada item da lista usamos a tag <a></a>. Onde no parametro href o valor será o nome do arquivo da pagina que queremos acessar nesse link. Como a pagina está na mesma pasta que o index, não necessita do endereço completo, apenas o nome do arquivo. Para indicar que essa página segue para uma proxima o parametro rel tera o valor next e na pagina seguinte tera o valor prev.

Essa pagina é um oferecimento UOL . Usamos o paramentro rel com valor nofollow, para que a busca do google não siga esse link como sendo da nossa página

Você também pode acessar nossa pagina de noticias

Para acessar uma pagina que esteja em uma pasta no mesmo local do index, no parametro href colocamos o nome da pasta uma barra invertida(/)e ai sim o nome do arquivo. Já na pagina que abriu para voltar precisamos no href colocar ../ antes no nome do index( ou qualquer outra pagina que esteja fora dessa pasta e no mesmo local que o index). Também informar que a pagina abrira no lugar da pagina aberta com o valor _self no target.
