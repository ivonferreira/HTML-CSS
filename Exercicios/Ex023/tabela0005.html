Definindo configurações para colunas

Para definir configurações para colunas de tabelas o modo dificil é vc ir em cada th ou td na mesma coluna e definir uma classe para ela. Mas no html5 existe uma forma mais simples de fazer isso, basta criar uma tag colgroup e dentro dela tags col na quantidade de colunas que a tabela tem (ou voce pode usar a tag span para expandir para a outras colunas, de forma que todas as colunas tenham uma representação da tag col). Nessa tag col nós aplicamos os estilos.
