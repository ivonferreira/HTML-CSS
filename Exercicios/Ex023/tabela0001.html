Minha primeira tabela em HTML

Hierarquia de tabelas (simples) Table = tabela
Table header = Cabeçalho de tabela
Table data (td)= Dados de tabela Table row (tr)= Linha Para construir tabelas usamos a tag Table e dentro dela usamos a tag tr para construir as linhas e dentro td para os dados que serão as colunas. Podemos definir estilo para cada linha, coluna ou celula através da declaraçao da table, tr ou td usando as classes e ids. A tabela criada não terá linhas separatorias, então definimos como bordas(border) para as celulas. De forma padrão as celulas terão uma separação entre elas. Para que não se separem usamos o parametro border-collapse e valor collapse. O fechamento da tag td é opcional, mas é recomendado. Se a hieraquia não for respeitada os dados apareceram antes ou após as tabela.

Alinhando o conteudo

Para o alinhamento dos dados na tabela nos configuramos o td. Para alinhamento horizontal usamos o text-align como no texto em geral. Para o alinhamento vertical usamos o parametro vertical-align onde podemos alinhar por top, bottom e middle(padrão).
