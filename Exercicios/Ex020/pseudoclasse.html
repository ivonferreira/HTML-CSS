Pseudo-classes

São caracterista que são aplicadas em uma determinada condição. Usamos o nome do elemento dois pontos(:) e a condição que queremos que essa formatação aconteça, e dentro declaramos como será quando essa ação acontecer. Ex div:hover{background-color: red;}, nesse caso hover é quando passamos o mouse emcima do objeto.

Quando queremos declarar uma tag que esteja dentro de outra usamos o maior que (>) da tag pai para a tag filha. Ex. div > p {color:red;}

Algumas pseudo-classes

  • hover - Quando passar o mouse
  • active - Quando o link é clicado
  • visited - Quando o link já foi aberto
