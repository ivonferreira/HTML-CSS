Trabalhando com listas

Para cada item da lista usamos a tag <li></li>

Listas Ordenadas

Listas em que a ordem é importante. Para esse tipo, a lista deve estar entre as tags <ol></ol>

Podemos variar os ordenadores para: numeral(1), letras(a,A) e romano(i,I). atraves do atributo type, e qual o ponto inicial pelo atributo start.

  1. Acordar
  2. Ligar para o Trabalho
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a Faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Listas não ordenadas

Listas onde não há necessidade de ordem. e para cada item da lista. Usamos a tag <ul></ul>.

Podemos variar o tipo (type) para esses valores:
  • Disc
  • Circule
  • Square

Lista de Definição

Listas que contém um termo e uma definição. e para cada item da lista.Usamos a tag <dl></dl> para o termo a tag <dt></dt> e para a definição usamos a tag <dd></dd>.

Termo
Definição

Linguagem de marcação para a criação de conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
