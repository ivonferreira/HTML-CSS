Identificando elementos HTML para estilos

Para criar uma formatação especifica para alguns determinados elementos da pagina usamos o parametro id e nomeamos esse id. No arquivo de estilo indicamos o elemento com uma tralha(#) e o nome do id então dentro declaramos caracteristicas que só funcionariam nesse id.Também podemos declarar só a tralha e o id(mas ai qualquer elemento que tenham esse mesmo id será aplicado). Ex id#princiapal {background-color: blue;} ou #principal {text-align:center};

O limitador do id é que somente um elemento pode usar o identificador

Se em HTML é id na CSS usamos #, se usamos em HTML class na CSS usamos. Apesar de funcionar, é considerada uma forma errada de usar o id pelo W3C. Para suprir essa necessidade usamos o atributo class e no CSS declaramos os estilos como o id só que ao invés do # usamos o ponto(.). Obs: não nomear class e id pela caracteristica(nome da cor, tamanho, font) nomeie pela função(Principal, segundo, destacado etc.), pois se caso for preciso mudar a caracteristica o nome perde o sentido.

Podemos adicionar a qualquer parte do texto uma tag chamada span e nela indicar a class que criamos anteriormente, então nesse parte podemos usar sua formatação.

Podemos adicionar mais de uma class para um elemento basta entre as aspas colocar o nome das duas classes com um espaço entre elas. Podemos usar class e id no mesmo elemento também. nesse caso o id tem a prioridade.
