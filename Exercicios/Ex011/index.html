Imagem Dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente ou diminua o tamanho do seu navegador

Para uma imagem dinamica usamos a tag <picture>e dentro dela colocamos a <img> com a imagem em tamanho grande, acima adicionamos a tag <source> alteramos para de min-width para max-width e colocamos o tamanho da maior imagem mais 50px, no parametro scrset indicamos o arquivo de tamanho medio e no type será image/o tipo do arquivo.

Para adicionar uma imagem de tamanho menor, colocamos acima da anterio obrigatoriamente, a mesma tag source, indicando o arquivo menor, e o max-width será o tamanho da imagem media + 50px, não esquecendo de preencher o type como o anterior. Podemos adicionar vários tamanhos de imagem, respeitando a ordem e o tamanho da janela ou tela.

Reproduzindo Audio

Vamos aprender a reproduzir audios em HTML5

Para usar audio em HTML5 usamos a tag <audio> de duas formas:

  • A tag <audio> com um parametro src indicando o arquivo mp3, ogg ou wav
  • A tag <audio> sem o src e dentro da tag colocamos a tag <source> com o paramentro src indicando o arquivo do audio e o type audio/mpeg(para mp3)ogg ou wav. Dessa forma podemos adicionar varias sources com formatos diferentes, e ele assim como nas imagens ele precisa de uma ordem,(o de cima primeiro) caso o primeiro não funcione , o segundo caso não, o terceiro e assim por diante. No ultimo colocamos o arquivo para download atraves de um link e uma mensagem de que o navegador não reproduz o audio

Os seguintes parametros podem ser adicionados nos dois tipos de tag <audio>

  • autoplay- Nem sempre funciona, mas serve para reproduzir o audio ao carregar o site
  • preload- Se o audio deve ser carregado antes do site aparecer
    • auto - Carrega junto com o site, é o mais perigoso pois o site só vai aparecer quando o audio estiver totalmente baixado
    • metadata -Carrega junto com o site apenas as informações basicas do audio, como duração, nome, velocidade
    • none - Carrega só após o carregamento do site
  • controls - Exibe um controlador de audio básico
  • loop - Assim que o som terminar de ser tocado ele começa novamente, usado para musicas de fundo e outros
