Exemplo de formulário

Dados Pessoais:

Para email usamos o input email. E o telefone input tel. Para indicar o formato do telefone usamos expressões regulares. Nessa situação usaremos no pattern ^\(\d{2}\)\d{4,5}-\d{4}$ no formato (XX)XXXX-XXXX

Para agrupar um conjunto de informações usamos a tag fieldset e dentro uma legend. Dessa forma o formulário fica mais organizado e podemos modificar através do css.
