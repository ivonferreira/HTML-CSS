Exemplo de formulário

Outros input são o color, que indica uma cor e ele envia o codigo hexadecimal, para já indicar uma cor padrão é so indicar no value.Mais um input é o range esse define um nivel, podemos indicar o maximo e o minimo e o step, ele cria uma barra que indica um valor e podemos indicar um value padrão também. Dependendo do navegador podem não ser compativeis. O input file serve para enviar um arquivo, geralmente uma foto de perfil, já que estamos enviando um arquivo ele deve ser usado no metodo post.
