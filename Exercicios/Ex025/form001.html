Meu primeiro formulário

Para criar um formulário, usamos a tag form e nela indicamos os campos que serão usados, ao lado dos campos usamos um input de texto com nome e id. Mas a principio o titulo do formulário e o input não possuem nenhuma ligação para isso no texto que indica o que entrará naquele input usaremos a tag label e indicamemos no for o id do input. Todo input de texto terá nome e id. O nome é usado mais em Php e html e o id é no JavaScript.

Caso você não queira que o formulário tenha o autocompletar usamos no form uma preferencia chamada autocomplete com valor off.

No final precisamos de uma entrada para submeter as informações digitadas, para isso criamos um input do tipo submit que precisa de um value .

Existem de 2 metodos (method)de envio de formulário: GET é o padrão se não for especificado será esse o método, o seja se os dados não forem sensiveis ou para compartilhar os dados que foram usados, usamos esse metodo. POST dessa forma os dados não aparecem no endereço apesar dos dados enviados não aparecerem eles estão apenas escondidos mas são acessiveis.

Para enviar formulários no metodo get só podemos enviar no máximo 3000 bytes ou seja 3000 letras e com envio de outros arquivos de imagens e outros. Caso contrário usaremos o POST.

Para criar um campo de senha usamos no input, ao inves de type text, usamos type password. E podemos adicionar um input para resetar os campos no caso o type será reset

Para selecionarmos quais campos do formulários são obrigatórios para que seja enviado usamos o parametro required. Para requerer uma quantidade minima e máxima de caracteres usamos os parametros minlength e maxlength.Outros parametros size tamanho do campo em caracteres, placeholder o texto que aparece no campo antes de digitarmos algo.

Para o autocomplete ao indicamor no form, teremos apenas 2 opções on e off. Caso indiquemos no campo precisamos indicar qual o tipo de informação que queremos que ele faça a sugestão, no caso de senha temos 2 opções new-password (para sugerir uma nova senha) e current-password (para sugerir uma senha já utilizada.
