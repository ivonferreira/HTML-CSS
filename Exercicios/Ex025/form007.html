Formulários sem input


Uma forma de criar uma lista de opções num formulário usamos o select e dentro dessa tag usamos as opções sem tas option com o value e o nome. Podemos também agrupar um conjunto de options com a tag optgroup e usamos uma parametro chamado label para dar o nome do grupo. Para deixar uma opção padrão indicamos o parametro selected no option escolhido.Para deixar uma opção como um placeholder criamos uma opção com value vazio e colocamos ela como selected.

Podemos adicionar listas de opções em inputs que nos digitamos semelhantes ao autocomplete. Para isso após o input usamos a tag datalist e dentro inserimos as opções (com ou sem value) e no datalist indicamos uma id e no input acrescentamos um atributo list e no valor o id da datalist.

Podemos também deixar uma area para que seja digitado um texto livre, para isso usamos a tag textarea nela devemos indicar o nome , id, colunas e linhas. As linhas e colunas são o tamanho na pagina.Não há limite de texto que pode ser digitado, por isso devemos usar o metodo post.
