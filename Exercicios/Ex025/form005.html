Para o input checkbox usamos o label apos a tag, e indicamos o id da mesma forma. Para o input de opção unica usamos o radio, e nesse caso os name de todas as radios precisam ser igual e id diferente, pois só um valor será aceito. Para que uma opção já esteja marcado de forma padrão usamos o parametro checked, no caso do radio só um pode ficar marcado, caso mais de um esteja marcado ele só marcará o ultimo. Para que o radio ou checkbox envie um determinado valor usamos o value em cada opção
